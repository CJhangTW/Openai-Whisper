# Openai-Whisper

## 新增虛擬機

`py -m venv venv`

## 啟用虛擬工作環境

`.\venv\Scripts\activate`

升級pip套件
`py -m pip install --upgrade pip`

列出所有已安裝套件
`pip freeze`

常見相依性套件安裝方法
`pip install -r requirements.txt`

懶的話: `pip install -r requirements.txt > requirements.txt`

相依性套件輸出
`pip freeze > requirements.txt`

## 安裝 Whisper

`pip install openai-whisper`


## 使用 whisper 

範例:
`py .\App.py`
